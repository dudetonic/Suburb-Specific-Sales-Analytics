# Suburb-Specific-Sales-Analytics

### Project Overview
A data analysis and business intelligence project focused on transforming raw sales data into actionable, location-based insights. The primary goal is to provide a granular view of performance by tracking key metrics and trends across specific suburbs and local markets. The output is a multi-dimensional visualization suite (dashboards) designed for strategic resource allocation, sales forecasting, and identifying geographically-driven growth opportunities.

### Data Sources
Sales and Suburb Performance: This is the primary dataset used for this analysis is the "Sales_and_suburb_performance-excel" file.

### Tools
- Microsoft Excel
  - Used for data analysis and building the overall dashboard


 ### Exploratory Data Analysis
 EDA involved exploring the Sales and Suburb Performance to answer key questions, such as
 - Top Buyers in Each Year
 - Distinct Count of Sales
 - Suburb Groups from 2016 - 2017
 - Distinct Count of Manager
 - Distinct Count of Day

### Data Analysis
Identified trends and patterns in the dataset called "Sales_and_Suburb_performance-excel" file.

Visualization: Built visuals including KPI cards, bar charts, column charts and Line charts to present insight more clearly
